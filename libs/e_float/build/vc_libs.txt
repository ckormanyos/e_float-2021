Microsoft Visual C++ x86 gmp build needs these libraries:
  ./e_float/e_float/gmp/4-2-4/vc9/p4/gmp.lib
  ./e_float/e_float/gmp/4-2-4/vc9/p4/gmp.pdb

Microsoft Visual C++ x64 gmp build needs these libraries:
  ./e_float/e_float/gmp/4-2-4/vc9/x64/gmp.lib
  ./e_float/e_float/gmp/4-2-4/vc9/x64/gmp.pdb

  ./e_float/e_float/gmp/4-2-4/vc9/p4/gmp.lib
